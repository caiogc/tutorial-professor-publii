Tutorial de utilização do Publii

Criando o site a partir de um tema

Os temas no Publii definem a aparência geral do seu site; eles decidem o layout de posts e páginas, e como a homepage se parece. O Publii vem com um tema pré-instalado para você e define o tema Simple padrão por padrão.

Abaixo você encontrará mais informações sobre como instalar os temas.

Nota: Se você já tem um site publicado, sempre faça um backup antes de instalar, ativar, desinstalar ou trocar temas no Publii, para garantir que poderá restaurar o estado anterior caso algo saia errado. Consulte nosso tutorial de backup para aprender como fazer isso.

Instalando um tema

Após o download ser concluído, você precisará instalar o tema por meio da interface Publii. Este é um processo muito simples. Para fazer isso, siga estas etapas:

  1. Abra o aplicativo Publii e clique no  ícone do menu de três pontos no canto superior direito da interface do Publii para abrir o menu principal e, em seguida, clique no botão Themes .
  2. Você verá uma lista dos seus temas instalados. Nesta página, clique no  botão Install Theme no topo. Isso abrirá o Explorador de Arquivos.
  3. Navegue até a pasta onde você baixou o  arquivo .zip do tema do Publii Marketplace e clique duas vezes nele.
  4. O Publii agora instalará o tema e o adicionará à lista de temas disponíveis.   

Seu tema agora está instalado.

Ativando o tema

Quando um novo tema é instalado no Publii, ele não adiciona automaticamente os arquivos de tema ao seu site; isso é para evitar que os arquivos do seu site fiquem inchados com arquivos de tema que não estão em uso. Para usar o tema no seu site ativo, siga estas etapas:

  1. No aplicativo Publii, clique em Site Settings no menu da barra lateral esquerda.
  2. Na tela Site Settings, clique na caixa suspensa ao lado da configuração Current Theme.
  3. Na lista de temas, você verá o tema recém-instalado na  seção Install and Use; selecione-o e clique no  botão Save settings na parte superior da tela.
  4. Seu novo tema agora está ativado; visualize seu site e você poderá ver como ele ficou!

Se você estiver satisfeito com seu novo tema e desejar continuar a usá-lo, é uma boa ideia  desinstalar o tema usado anteriormente para liberar espaço e reduzir o peso geral do seu site. Para fazer isso, clique na lista suspensa ao lado da opção Current Theme e, sob o  título Uninstall, selecione o tema que deseja desinstalar e clique no  botão Save Settings; os arquivos serão removidos agora.

Note que desinstalar o tema dessa maneira não o remove completamente do Publii; você ainda poderá voltar a ele a qualquer momento usando a  opção Install and Use. Desinstalar simplesmente impede que os arquivos do tema sejam carregados no seu site.

Mudando para um tema diferente

Trocar para um novo tema depois de ficar entediado com o atual é superfácil; basta abrir a tela Site Settings novamente e escolher seu novo tema na seção Install and Use da opção Current Theme. Se quiser experimentar um tema completamente novo, você pode usar o Publii Theme Marketplace para baixar e instalar um novo tema; isso aparecerá na seção Install and Use da seção Current Theme. Se você já trocou entre vários temas no seu site atual, notará que todos os seus temas instalados anteriormente aparecem sob um título Use separado ; isso permite uma troca rápida entre temas instalados sem a necessidade de copiar os arquivos novamente.

Desinstalando um tema

Se você alternou entre vários temas no seu site, os arquivos de cada tema serão incluídos nos arquivos do site que são gerados quando você sincroniza com seu servidor, potencialmente inchando um pouco o tamanho total do site. Para combater isso, você pode desinstalar quaisquer temas do seu site que não esteja usando; os arquivos ainda estarão disponíveis no Publii, eles simplesmente não serão incluídos nos arquivos do seu site.

Para desinstalar um tema, na seção Site Settings → Theme, abra a lista suspensa. Abaixo das seções Use e Install and Use, você verá uma terceira seção Uninstall, que exibe todos os seus temas instalados no momento. Clique no tema que deseja desinstalar e, em seguida, clique no botão Save settings para removê-lo dos arquivos do seu site. O tema ainda aparecerá na  seção Install and Use , então você poderá reativá-lo e usá-lo novamente a qualquer momento sem a necessidade de passar pelo processo de instalação novamente.
