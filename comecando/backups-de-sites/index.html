Tutorial de utilização do Publii

Backups de Sites

Perder os dados do seu site não é brincadeira, então vale a pena ter planos de contingência para recuperar seu site se o pior acontecer. Para ajudar você a manter seu site seguro, o Publii inclui um recurso de backup e restauração que permite salvar instantaneamente uma cópia de um site Publii em um arquivo compactado, pronto para ser carregado novamente mais tarde, se necessário.

Fazendo backup do seu site

Para criar um backup do seu site, clique na opção Tools no menu principal para visualizar as ferramentas disponíveis no Publii, depois clique na opção Backups no topo para ir para a tela de backups. Aqui você verá uma lista de backups disponíveis, que obviamente estarão vazios se você ainda não criou o seu primeiro. Para fazer um novo backup do seu site atual, clique no botão Create backup no canto superior direito da seção; um pop-up será aberto permitindo que você insira um nome para o seu backup; o padrão é um nome exclusivo com o formato backup-data-hora , mas você pode inserir qualquer nome que desejar e clicar em OK . Após alguns momentos, seu novo backup será criado e adicionado à lista.

Restaurando seu site

Para restaurar um backup existente do seu site, clique na opção Tools no menu principal, depois clique na opção Backups para abrir a tela de backups. Na lista de backups disponíveis, encontre o que você deseja usar e clique no botão Restore à direita da listagem; um pop-up aparecerá pedindo a confirmação de que você deseja restaurar a partir deste backup. Verifique novamente se o arquivo que você está restaurando está correto, pois assim que o processo for iniciado, todos os arquivos serão substituídos e a restauração não poderá ser revertida. Quando estiver pronto, clique em OK se desejar continuar; o backup será carregado e uma mensagem de confirmação aparecerá assim que o site for restaurado.

Restaurando backups em uma nova instalação do Publii

Se você mudou para um novo computador e quer restaurar o backup do seu site para uma nova instalação do Publii, você pode fazer isso na  tela Create New Website. Ao abrir o Publii pela primeira vez, você verá automaticamente esta tela na inicialização. Em vez de preencher o nome do site e outros detalhes, clique na  aba Install From Backup na parte superior do pop-up, ao lado do  texto Create New Website:

Uma nova caixa aparecerá onde você pode arrastar e soltar seu arquivo de backup, ou clicar no  botão Choose File para abrir o explorador de arquivos para escolher seu arquivo. Uma vez que o arquivo é selecionado, uma caixa de diálogo será aberta onde você pode digitar o nome do site; este será o padrão para o nome do site original, mas você pode alterá-lo se desejar. Em seguida, clique no  botão Create Website para restaurar seu site; uma vez que o processo for concluído, o site restaurado será aberto automaticamente, pronto para modificação.

Se desejar restaurar um site de um backup para uma instalação existente do Publii, você pode acessar a tela Criar novo site clicando no ícone de seta para baixo  à direita do nome do site  na parte superior do menu à esquerda no aplicativo Publii para abrir a lista de sites e, em seguida, clicando no botão Create New Website na parte superior direita do aplicativo.
