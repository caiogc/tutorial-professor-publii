Tutorial de utilização do Publii

Download e instalação

Como o Publii é um aplicativo de desktop em vez de um CMS do lado do servidor, a instalação é o mais fácil possível. Primeiro, você precisará pegar o aplicativo na Seção Oficial de Download do Publii ; há downloads disponíveis para versões do Publii para Windows, Mac e Linux. Escolha seu SO e clique no botão de download! Quando o download estiver concluído, clique duas vezes no arquivo baixado para executar o instalador.

Uma vez feito isso, o Publii será iniciado automaticamente; para uso futuro, um atalho será adicionado à área de trabalho e à pasta do programa se você estiver usando Windows ou Linux, ou ao Launchpad no Mac para que você tenha acesso fácil ao Publii na próxima vez.

Para Windows

Em sistemas baseados em Windows, os arquivos de aplicativo, que controlam a interface e as funcionalidades do aplicativo, são armazenados no local de instalação do Publii (normalmente C:\Users\USER NAME\AppData\Local\Publii). Esses arquivos não devem ser modificados, exceto por desenvolvedores experientes, caso contrário, você pode causar travamentos ou outros problemas com o aplicativo.

Os arquivos de conteúdo para Publii, que contêm os arquivos para os sites e outros conteúdos que o Publii gera, são armazenados separadamente na área Documentos para o usuário atual, por exemplo C:\Users\USER NAME\Documents\Publii. Esses arquivos incluem pastas para os temas instalados no momento, pastas para cada site gerenciado atualmente pela instância instalada do aplicativo Publii (contendo todos os arquivos-chave, tags, dados de postagens etc.) e arquivos de configuração que armazenam as configurações de opções atuais para o Publii. Esses arquivos podem ser modificados por desenvolvedores web com experiência em arquivos .json, .hbs e .css para modificar elementos principais de um tema, as configurações do Publii ou sites criados.

Para MacOS

Assim como no Windows, o Publii separa seus arquivos no MacOS em dois tipos, os arquivos de aplicativo, que controlam o aplicativo e suas funcionalidades, e os arquivos de site, que contêm todo o conteúdo que o Publii usa para criar sites. Os arquivos de aplicativo são armazenados na área padrão para aplicativos, /Applications/Publii. Eles não precisam ser modificados, exceto por desenvolvedores web experientes que buscam modificar as funções principais do aplicativo Publii; fazer alterações nesses arquivos pode resultar em travamentos ou outras instabilidades.

Os arquivos de conteúdo, que abrangem pastas contendo os arquivos dos temas instalados, conteúdo do site, como tags, posts e imagens, e arquivos de configuração contendo as configurações ativas na instância instalada do Publii, são armazenados em um local separado; /Users/USER NAME/Documents/Publii/. Esses arquivos afetam o site que o Publii gera, em vez do aplicativo em si; mais uma vez, eles devem ser modificados apenas por usuários com experiência em arquivos .css, .json e .hbs, como desenvolvedores de temas ou web designers.

Para Linux

A versão Linux do Publii armazena os arquivos de aplicativo que fornecem a funcionalidade e a interface do aplicativo na  /opt/Publii/resources/app/ pasta. Usuários que desejam fazer alterações na funcionalidade principal do aplicativo podem fazer suas alterações nesses arquivos, embora seja necessário ter cuidado; fazer tais alterações pode resultar em instabilidades ou travamentos.

Os arquivos de conteúdo, que armazenam o conteúdo real do site, incluindo tags, posts e imagens, bem como temas, além dos arquivos de configuração que armazenam as configurações individuais do aplicativo do usuário, são armazenados em um local diferente;  /home/USERNAME/Documents/Publii. Usuários com experiência em desenvolvimento web que desejam fazer alterações nos temas, ou ter acesso mais direto ao CSS do site, podem fazê-lo por meio dos arquivos armazenados neste local.
