Tutorial de utilização do Publii

Bem-vindo ao Tutorial de Publii

Este tutorial foi criado para ajudá-lo a entender como usar o Publii, uma ferramenta para criação de sites estáticos. Publii permite desenvolver sites de forma simples, sem a necessidade de programação, oferecendo uma interface intuitiva e a possibilidade de publicar seu conteúdo de maneira rápida e eficiente.

Aqui, você encontrará as instruções para instalação, configuração e uso do Publii, além de dicas para personalizar seu site e publicar online. O objetivo é tornar o processo de criação e gerenciamento de sites mais fácil, sem complicações.

O que você vai aprender

Neste tutorial, você encontrará todas as informações necessárias para configurar e gerenciar um site no Publii. O conteúdo está organizado em categorias para facilitar o aprendizado:

Configuração Inicial

  • Download e Instalação: Saiba como instalar o Publii e configurar a plataforma no seu computador.
  • Criando o Site a partir de um Tema: Aprenda a usar o tema disponibilizado pelo NTI da UFOP. Este tema é obrigatório para manter o padrão institucional.
  • Criando e Restaurando Backup: Descubra como criar backups para proteger seus dados e como restaurá-los, caso necessário.
  • Configuração do Servidor: Aprenda a configurar o servidor para publicar seu site de forma eficaz, com foco em FTP.

Editores de Texto

  • Block Editor: Entenda como criar e organizar conteúdos utilizando um editor baseado em blocos.
  • Markdown Editor: Explore o uso de Markdown para criar conteúdos de forma simples e rápida.
  • WYSIWYG Editor: Saiba como editar visualmente seus textos com um editor intuitivo, sem precisar de conhecimentos técnicos.
  • Inserindo Arquivos: Descubra como adicionar imagens, PDFs e outros arquivos ao seu site.

Gerenciamento de Conteúdo

  • Informações Básicas: Veja como personalizar informações essenciais do site, como foto do perfil, biografia e dados de contato.
  • Página Inicial: Descubra como personalizar o conteúdo da sua página inicial.
  • Postagens: Aprenda a criar, organizar e editar postagens no seu site.
  • Páginas: Descubra como criar e gerenciar páginas estáticas, como "Sobre" ou "Contato".
  • Tags: Utilize tags para organizar e categorizar seu conteúdo.
  • Menus: Personalize e organize os menus de navegação do site para facilitar a experiência dos visitantes.

Autoria e Colaboração

  • Autores: Veja como adicionar e gerenciar múltiplos autores no seu site, atribuindo permissões específicas.

Sincronização com o Servidor

  • Sincronizando o Site com o Servidor: Entenda como sincronizar as alterações feitas no Publii com o servidor de hospedagem para manter seu site atualizado.

Com este tutorial, você estará preparado para criar e gerenciar um site completo utilizando o Publii. Clique em qualquer uma das seções acima para começar a aprender e aproveite todos os recursos disponíveis!
