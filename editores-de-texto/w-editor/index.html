Tutorial de utilização do Publii

WYSIWYG editor

Como quase qualquer outro CMS, o Publii inclui um editor WYSIWYG (What You See Is What You Get) para criação de conteúdo. Este editor permite que os usuários escrevam texto, insiram imagens e outros elementos, assim como você faria em um processador de texto padrão como o Microsoft Word, e os converte em HTML amigável ao navegador para você. Como tal, você descobrirá que escrever conteúdo é fácil, mesmo que nunca tenha criado um site antes, mas há algumas opções adicionais que vale a pena conhecer, que estão disponíveis na barra de ferramentas do editor, que veremos aqui. 

A barra de ferramentas do editor WYSIWYG

Como qualquer processador de texto padrão, o editor WYSIWYG inclui opções para formatação de texto comum, bem como funções mais específicas para sites. Nesta seção, abordaremos cada uma das opções da barra de ferramentas. Se você não tiver certeza do que um ícone em particular faz, basta passar o mouse sobre ele para ver uma dica de ferramenta. 

Formatação básica

Existem algumas opções que nenhum processador de texto deve ficar sem, e a primeira seção da barra de ferramentas abrange estas:

  • Alterar texto para  negrito.
  • Alterar texto para  itálico.
  • Adicionar sublinhado ao texto.
  • Texto tachado.
  • Alinhar texto à esquerda.
  • Alinhar o texto ao centro.
  • Alinhar texto à direita.

Nesta primeira seção, também há algumas opções que são menos comuns em processadores de texto, como  Inserir/Remover Link,  Inserir Imagem,  Inserir Vídeo  e  Criar Índice, bem como opções que são comuns em processadores de texto, mas que têm opções adicionais ou funcionam de forma ligeiramente diferente, como  Listas de Marcadores e Listas de Números. Informações sobre essas opções podem ser encontradas abaixo. 

Inserindo e removendo links

Para inserir um link, mova o cursor para a área onde você quer adicionar o link, ou realce o texto que você quer transformar em um link, então clique no  ícone Inserir Link na barra de ferramentas. Isso abrirá um menu pop-up com opções de link adicionais, como segue:

  • Url - Insira a URL da página para a qual você quer que o texto faça o link aqui. Se você quiser criar um link de âncora para outro título na mesma página, clicar neste campo exibirá uma lista de títulos disponíveis, bem como os links padrão  superior  ou  inferior.
  • Text to display - Insira o texto que deve ser transformado em um link aqui; se você destacou um bloco de texto antes de clicar no ícone, ele já estará preenchido.
  • Title - O texto inserido aqui será exibido quando o usuário passar o mouse sobre o link. 
  • Target - Se definido como None, o link será aberto na mesma janela do navegador. Se definido como  New Window, o link será aberto em uma nova aba ou janela, dependendo das configurações do usuário.

Clique em  Ok quando os campos forem preenchidos para inserir o link. Para remover um link, basta clicar no texto vinculado no editor e, em seguida, clicar no  ícone Remove Link na barra de ferramentas.

Inserindo Emoticons

O editor Publii WYSIWYG inclui uma seleção extensa de emoticons que podem ser inseridos em seu conteúdo. Para abrir a tela de emoticons, clique no  ícone Emoticon (que parece um rosto sorridente) na barra de ferramentas. Isso abrirá um pop-up com os emoticons disponíveis:

A partir daqui, você pode selecionar qual emoticon você gostaria de inserir em seu conteúdo. As categorias à esquerda do pop-up permitem que você filtre os emoticons por assunto, ou você pode usar a  barra de pesquisa na parte superior do pop-up para procurar um emoticon específico. Quando você encontrar o emoticon que deseja, clique nele para inseri-lo em sua publicação.

Inserindo Citações em Bloco

Blockquotes são uma ótima maneira de destacar partes importantes do texto em sua postagem ou chamar a atenção para citações específicas. Para inserir um blockquote, clique no  ícone Blockquote na barra de ferramentas. Isso centralizará seu texto, aumentará o tamanho da fonte e adicionará um ícone antes do texto para indicar que é uma citação. Você pode então continuar inserindo texto normalmente; pressionar  Enter uma vez moverá o cursor uma linha para baixo, mas preservará o estilo do blockquote. Pressione Enter novamente sem adicionar mais texto para encerrar o blockquote.

Inserindo listas de marcadores e números

Publii inclui os dois estilos básicos de lista que podem ser encontrados em editores de texto; listas de marcadores e de números. Eles são essencialmente os mesmos, mas as listas de marcadores exibem cada item com um único marcador, tornando-as úteis ao inserir uma lista de coisas que não estão ordenadas, enquanto as listas de números usam números ascendentes em vez de marcadores, tornando-as melhores para instruções que precisam ser seguidas em ordem. 

Para inserir uma lista de qualquer tipo, coloque o cursor onde você quer que a lista seja inserida e clique no ícone de marcador ou lista numérica. Isso criará o primeiro item na lista. Insira o texto para o item e pressione Enter quando terminar para criar automaticamente o próximo item. Depois que todos os itens forem adicionados, pressione Enter em um item em branco para remover o espaço em branco e encerrar a lista. 

Um recurso adicional no editor do Publii é a capacidade de escolher um estilo específico de número ou marcador; clicando na seta à direita dos  ícones da Lista de Inserção,  você pode escolher manter o estilo padrão ou alternativas como algarismos romanos, discos ou quadrados.

Inserindo Imagens

Existem duas maneiras de adicionar imagens às suas postagens. A primeira, e mais fácil, é simplesmente arrastar e soltar a imagem que você deseja usar na área do editor; a imagem será carregada automaticamente e inserida na posição do cursor selecionada no momento. No entanto, esse método só insere a imagem com as configurações padrão. Se você quiser um controle mais fino sobre a aparência das imagens, você deve usar o  botão Inserir/Editar Imagem, que está no canto superior direito da barra de ferramentas do editor.

Clicar nele abrirá a caixa Inserir/Editar Imagem, onde você verá um conjunto de cinco opções que permitirão que você faça alterações na aparência da imagem. Essas opções são:

  • Source - Exibe o caminho para a imagem que será inserida no post. Clique no ícone à direita do campo para abrir o navegador de arquivos e selecione a imagem que deseja inserir do seu computador. Alternativamente, se a imagem que deseja usar estiver localizada online, você pode inserir a URL da imagem neste campo; lembre-se de garantir que você tenha a permissão necessária para postar uma imagem se não possuir os direitos sobre ela!
  • Alternative Description - Adicione uma descrição à imagem; ela será exibida caso a imagem não carregue ou será lida por um leitor de tela para fornecer informações sobre a imagem para deficientes visuais.
  • Width/Height - Define a largura e a altura da imagem em pixels; quando uma imagem é selecionada, as dimensões reais serão preenchidas automaticamente nesses campos, embora você possa alterá-las para redimensionar a imagem, se necessário. 
  • Restringir Proporções (Ícone de Cadeado) - Quando esta opção estiver habilitada, a proporção da imagem escolhida será preservada ao redimensionar, garantindo que a imagem não pareça muito esticada. Uma vez habilitada, inserir uma largura no campo Largura  preencherá automaticamente o campo altura com o tamanho apropriado.
  • Class - Esta opção permite que você modifique a aparência da imagem no post. Escolha entre  Full Image, que dimensiona a imagem para caber na coluna de texto inteira,  Wide Image, que estica a imagem até as bordas da página, ou  Left, Right ou Center-Aligned Image, que coloca uma versão menor da imagem à esquerda, direita ou centro da tela para imitar um layout de jornal.
  • Show Caption - Quando esta opção estiver ativada, uma caixa de legenda será adicionada abaixo da imagem inserida no editor, onde você poderá inserir o texto da legenda para a imagem.

Nota:  Se você estiver encontrando problemas ao carregar, modificar ou gerar imagens, tente alternar do  mecanismo de imagem Sharp padrão  para o  mecanismo JIMP incluído, que pode ser mais estável para você. Você pode alterar qual mecanismo é usado para modificar imagens na  seção Configurações globais (opções disponíveis somente para usuários de Windows e MacOs).

Editando uma imagem existente

Se você inseriu uma imagem anteriormente em sua postagem e deseja modificar algumas de suas configurações, você pode fazê-lo clicando primeiro na imagem que deseja modificar no editor, depois clicando no botão Inserir/Editar Imagem. Isso reabrirá a caixa pop-up contendo as opções de imagem, que você pode então modificar de acordo.

Inserindo Galerias

As galerias no Publii são alimentadas pelo Photoswipe, uma solução leve que cria galerias de imagens sensíveis ao toque com o mínimo de esforço.

Criando uma galeria

  1. Para adicionar uma galeria a um post, clique no ícone que exibe várias imagens (à direita do ícone padrão Inserir Imagem). Uma seção representando a galeria será adicionada ao corpo do post definido por um contorno, com um botão Add images to gallery no centro.
  2. Clique no botão Add images para abrir um pop-up Edit gallery; se esta for uma nova galeria, uma janela do explorador de arquivos também será aberta, onde você pode selecionar quaisquer imagens para adicionar à sua galeria.
  3. Clique duas vezes em uma imagem para adicioná-la à sua galeria; depois que ela for adicionada, você poderá adicionar um texto alternativo e uma legenda de imagem por meio dos campos fornecidos.
  4. Para adicionar mais fotos, clique no botão Add images no canto inferior direito da janela de edição.
  5. Depois que todas as suas imagens forem adicionadas, selecione o número de colunas para a galeria usando a caixa suspensa no canto inferior esquerdo da janela de edição e clique no botão Ok para salvar as alterações.

Editando uma galeria

Para editar uma galeria, abra a publicação que contém a galeria que você deseja editar e clique nela no editor WYSIWYG; isso reabrirá o pop-up Edit gallery para que você possa adicionar ou remover fotos, editar o texto alternativo ou as legendas, o número de colunas e alterar o alinhamento de largura larga/largura total.

Personalizando sua galeria

Você pode personalizar o estilo das suas galerias para ajudá-las a se adequarem à estética do seu site e definir quais controles os usuários poderão acessar. Cada uma dessas opções é aplicada globalmente a todas as galerias do seu site e pode ser encontrada clicando na opção Theme na barra lateral esquerda e, em seguida, clicando na guia Galeria na seção Custom Settings da página Configurações do tema. As opções disponíveis são:

  • Espaçamento entre itens - Define a quantidade de espaço em branco entre cada imagem da galeria, cujo padrão é 0,5 REM; os valores REM são recomendados para manter o espaçamento relativo em diferentes tamanhos de tela (basicamente, o tamanho do espaço entre as diferentes fotos permanece relativamente o mesmo em todos os tamanhos de tela, portanto, não há grandes lacunas em telas pequenas), mas outros, como px e vw, podem ser usados.
  • Estilo do lightbox - Defina se a caixa de luz (o fundo mostrado ao redor de uma foto quando você clica nela na galeria) é de cor clara ou escura, para se adequar à estética do seu site.
  • Transparência de sobreposição do lightbox - Define a transparência da caixa de luz ao visualizar uma única foto; uma configuração de 0 será totalmente transparente, enquanto 1 será totalmente opaco.
  • Configurações avançadas do lightbox - Quando ativadas, opções adicionais para controlar seu lightbox serão mostradas.

Se as configurações avançadas do lightbox estiverem habilitadas, as seguintes opções adicionais estarão disponíveis, fornecendo mais controles no lightbox, além dos controles de toque padrão:

  • Slide atual/número total - Se ativado, o número total de imagens na galeria, bem como o número da foto visualizada no momento, serão exibidos na caixa de luz.
  • Botão de zoom in/out - Controla se um botão separado para zoom in e out será adicionado à caixa de luz; os usuários também podem aumentar ou diminuir o zoom clicando ou tocando na imagem.
  • Alternar botão de tela cheia - Adiciona um botão para tornar uma imagem em tela cheia para uma visualização mais detalhada.
  • Botão de compartilhar - Habilita um botão de compartilhamento para que os usuários possam compartilhar suas fotos nas redes sociais.
  • Botão de fechar - Exibe um botão para fechar a caixa de luz ao visualizar uma única foto.
  • Setas de próximo/anterior - Habilita botões para rolar entre fotos em uma galeria.
  • Pré-carregador - Habilita uma animação de pré-carregador que será ativada sempre que as imagens demorarem mais de 1s para carregar.
  • Fechar galeria ao rolar a página - Se habilitado, o lightbox da galeria será fechado quando o usuário rolar a página para baixo (observe que essa opção só se aplica a dispositivos que não suportam toque, como desktops).
  • Navegação por teclas de seta - Permite que usuários de desktop ou laptop usem o teclado para navegar entre as imagens na galeria.

Inserindo vídeo

Vídeos online fornecidos via Youtube, Dailymotion e similares são uma forma popular de fornecer e consumir conteúdo, então o Publii inclui suporte para incorporar esses vídeos em suas postagens. Para inserir um vídeo, assim como ao inserir uma imagem, coloque seu cursor no editor onde você quer que o vídeo seja inserido, então clique na opção Inserir/Editar Vídeo no canto superior direito da barra de ferramentas do editor; isso abrirá a caixa pop-up com as opções específicas do vídeo, que são:

  • Source - A URL de origem do vídeo que você deseja incorporar; basta copiar a URL completa (por exemplo, http://www.youtube.com/watch?v=11a11a11a11 ) e adicioná-la a este campo; o código de incorporação necessário será gerado automaticamente. Como alternativa, você pode copiar o código de incorporação fornecido diretamente da página do vídeo e inserir o código completo na aba Embed, disponível clicando no texto Embed na parte superior da caixa.
  • Width/Height  - Defina a largura e a altura do vídeo como você quer que ele apareça na postagem, em pixels.
  • Restringir proporções  (ícone de cadeado) - Assim como ao inserir imagens, habilitar esta opção garantirá que a proporção do vídeo seja preservada quando redimensionado; uma vez habilitada, inserir uma largura no campo Largura preencherá automaticamente a altura para corresponder à proporção original.

Você também deve ver três seções diferentes listadas no lado esquerdo do pop-up; Geral (a seção padrão), Embed (permite que o código de incorporação para seu vídeo seja adicionado em vez de especificar uma fonte direta) e Advanced. As opções avançadas permitem duas configurações adicionais:

  • URL de fonte alternativa - Se você tiver um local alternativo que o Publii possa usar caso a primeira fonte não funcione, insira-o aqui.
  • Pôster de mídia (URL da imagem) - Defina a miniatura do vídeo como ele será exibido no blog. Você pode inserir uma URL para a imagem que deseja usar ou clicar no  ícone de seta para cima à direita do campo para carregar sua própria imagem para ser usada como miniatura.

Editando as configurações de um vídeo

Se você já inseriu um vídeo e deseja alterar as configurações mais tarde, clique no vídeo no editor e depois clique no botão Adicionar/Editar vídeo para abrir a caixa que contém as configurações específicas para o vídeo selecionado, onde você poderá fazer as alterações necessárias.

Inserindo Tabelas

As tabelas no editor WYSIWYG funcionam de forma muito parecida com o que fazem em aplicativos como o Excel. É útil para exibir dados como preços, resultados de pesquisas etc...

Para inserir uma tabela, coloque o cursor no editor onde você gostaria de adicionar uma tabela, então clique no  ícone Tabela na barra de ferramentas, que abrirá as opções de tabela adicionais. Antes que sua tabela possa ser inserida em seu conteúdo, você precisará definir o tamanho; você pode fazer isso destacando a  opção Table, que exibirá uma grade. Destacando uma célula na grade com seu cursor, você pode definir o número total de linhas e colunas; clique na célula para definir o tamanho e inserir a tabela. 

A partir daqui, você pode clicar nas células individuais para inserir conteúdo. Você também pode clicar no  ícone Tabela novamente para definir opções de adição, como preenchimento e espaçamento de células, borda e cores, bem como executar tarefas básicas que você verá no Excel, como inserir linhas ou colunas adicionais, mesclar células e coisas do tipo. 

Criando um índice com links de âncora

O Post Editor inclui uma opção para gerar automaticamente um índice para seu post, o que pode ser útil para conteúdo de formato mais longo. Para inserir um índice, clique na seção do seu post onde você quer que o índice seja adicionado, então clique no  ícone do Índice correspondente , que é o ícone superior direito na barra de ferramentas do editor. 

A tabela inserida será gerada automaticamente a partir de cada um dos seus  títulos, com títulos aninhados sendo agrupados em subseções, por exemplo, o primeiro  título H2 será listado como seção 1 na tabela, com quaisquer  seções H3  abaixo dele aparecendo como seção  1.1, 1.2, 1.3 etc... e quaisquer seções H4 entre os  títulos H3  aparecendo como 1.1.1,  1.1.2 e assim por diante. 

Cada item no índice também terá um link âncora atribuído a ele, para que os usuários possam clicar em um item na lista para pular instantaneamente para essa seção na postagem. 

Alterando o título do índice

Por padrão, a tabela terá 'Table of Contents' como título. Isso pode ser alterado clicando no título da tabela no editor; você pode então excluir ou modificar o título conforme necessário.

Atualizando o Índice

Se mais seções e títulos tiverem sido adicionados à postagem após a geração do índice, você poderá atualizá-lo clicando no índice no editor e, em seguida, clicando no ícone circular Atualizar que aparece logo abaixo do último item na lista. 

Formatos

Cada tema no Publii inclui opções de formatação adicionais para destacar texto, além do  negrito ou itálico básicos. Esses formatos são acessíveis pela seção Formats no canto inferior esquerdo da barra de ferramentas do editor. Cada tema tem seus próprios formatos e estilos exclusivos; eles podem ser usados ​​para destacar blocos de texto com um fundo colorido, inserir uma letra maiúscula no início de uma frase ou deslocar texto. 

Para inserir o formato desejado, coloque o cursor do editor onde você quer que o formato seja usado, então clique na  opção Formats na barra de ferramentas e selecione o formato desejado na lista suspensa. Você pode então digitar normalmente, e a formatação será aplicada. Pressionar Enter encerrará a formatação e iniciará uma linha normal de texto.

Você também pode inserir formatação no texto existente destacando a seleção de texto, clicando na opção Formats e escolhendo um tipo de formato normalmente. 

Formatos HTML

A  opção Formatos HTML, que é a segunda opção a partir do canto inferior esquerdo da barra de ferramentas do editor, permite que você altere as tags HTML para o texto que você está escrevendo, incluindo  Cabeçalhos,  Parágrafos,  Endereço, Pré e  Código. Assim como nas outras opções de formatação, você pode selecionar o formato antes de digitar o texto ou destacar uma seleção de texto e alterar seu formato. Mais informações sobre os conceitos básicos de HTML podem ser encontradas na  seção Posts deste guia.

Localizar e substituir

Quando você precisa fazer muitas alterações de uma só vez, utilizar a função localizar e substituir economizará muito tempo. Para acessar a função  Localizar e substituir, pressione CTRL+F ou Command+F  no seu teclado, ou clique no ícone Lupa  na barra de ferramentas. Isso abrirá a caixa pop-up Localizar e substituir, onde você tem acesso às seguintes opções:

  • Localizar - Insira aqui o texto que você deseja encontrar no seu conteúdo.
  • Substituir por - Insira aqui o texto pelo qual deseja substituir o texto encontrado.

Clicar no  ícone de engrenagem no canto inferior esquerdo do pop-up também mostrará duas opções adicionais:

  • Diferenciar maiúsculas de minúsculas - Se esta opção estiver habilitada, a pesquisa procurará apenas por strings que correspondam exatamente às maiúsculas e minúsculas inseridas no campo Localizar; muito útil se você precisar alterar várias palavras minúsculas para maiúsculas ou vice-versa.
  • Encontrar apenas palavras inteiras - Se esta opção estiver ativada, a pesquisa ignorará o texto correspondente que faz parte de uma palavra maior, por exemplo, pesquisar por "up" com esta opção ativada ignorará palavras mais longas, como "chateado". 

Depois que as opções forem definidas, clique no botão Find na parte inferior do pop-up para pesquisar; todas as correspondências serão destacadas no seu texto, com a correspondência selecionada no momento destacada em azul. Você pode então escolher Substituir a correspondência única ou Substituir todas para alterar todas as correspondências. Para mover entre as correspondências, você pode clicar nas setas para cima e para baixo ao lado do campo Find.

Linha horizontal

Clicar no ícone Linha horizontal  inserirá um separador abaixo do texto selecionado no momento, indicando o fim de uma seção. O estilo da linha mudará dependendo da tipografia do tema que você estiver usando, mas a funcionalidade é sempre a mesma. 

Ler mais

Um read-more é uma ferramenta bem específica para escrita online. Alguns temas no Publii exibirão cada post em ordem, na íntegra, na sua homepage e páginas paginadas. Se você escreveu um post mais longo que preferiria que não fosse totalmente exibido na listagem da sua homepage, inserir um read-more ocultará qualquer texto depois dele, permitindo que os usuários que desejam ler o post completo cliquem na página do post individual. 

Nota:  A maioria dos temas Publii tem configurações para controlar o comprimento do texto de pré-visualização na página inicial, então os read-mores nem sempre são necessários. No entanto, pode ser útil adicionar readmores ao seu texto, independentemente de você planejar alterar seu tema posteriormente, pois seu novo tema pode não incluir essas configurações.

Limpar formatação

Clicar neste ícone removerá toda a formatação no texto selecionado. Isso pode ser usado se você acidentalmente usou negrito ou itálico em todo o texto, ou deseja retornar um bloco de texto à aparência padrão para reformulá-lo do zero.

Desfazer/Refazer alterações

Todo mundo comete erros, ou é pego por um comportamento inesperado do editor que bagunça sua formatação. Nesses casos, você pode clicar no botão Desfazer, representado por uma seta para trás, para reverter a alteração mais recente. Se você Desfazer algo, mas depois decidir que gostou da alteração, você pode clicar no  botão Refazer para reimplementar as alterações.

Postar salvamentos automáticos

O Publii inclui um recurso de salvamento automático para proteger sua postagem de qualquer fechamento inesperado, como queda de energia ou de uma simples mudança de ideia após fechar uma postagem. O salvamento automático ocorre a cada 10 segundos ao editar uma nova postagem; se você fechar a tela de postagem, seja intencionalmente ou devido a um fator externo, um rascunho da postagem salva automaticamente será salvo e mantido por 30 minutos, permitindo que você restaure quaisquer alterações perdidas. Para restaurar um rascunho, após reabrir a tela de edição de postagem, clique no botão Restaurar último rascunho que aparece à direita da barra de ferramentas do editor; ele tem um ícone que se parece com um relógio retrocedendo. Uma vez feito, o texto rascunhado aparecerá instantaneamente no editor, pronto para edição.

Editor de código fonte

Se você se sentir confortável usando HTML, ou quiser fazer algumas edições mais avançadas, como adicionar classes CSS ao conteúdo da sua postagem, você pode fazer isso alternando para o Editor de Código-Fonte .Clique no símbolo da tag HTML na barra de ferramentas do editor para alternar para o código-fonte. As funções do editor TinyMCE não estão disponíveis ao trabalhar com código-fonte, mas você pode pesquisar por conteúdo específico usando a barra de pesquisa, acessível clicando em CTRL + F.

Uma opção adicional que está disponível especificamente no editor de código-fonte é o botão Beautify Code; quando clicado, o código-fonte do seu post será formatado para facilitar a leitura e a modificação, em vez de para eficiência. Como alternativa, você pode configurar o editor para sempre embelezar o código sem a necessidade de clicar no botão todas as vezes; você pode fazer isso por meio do Three-Dot Menu. Este menu pode ser acessado clicando no  Three-Dot Icon  no canto superior direito do aplicativo (em qualquer página, exceto ao editar um post, tag ou menu) para abrir o menu principal e, em seguida, clicando na opção App Settings para abrir a  tela App Settings.

A partir daqui, role para baixo até a  seção Experimental Features e habilite a  opção Auto-Beautify Source Code in the WYSIWYG Editor e clique no  botão Save Settings no topo da página. Uma vez feito isso, o editor de código-fonte sempre embelezará o código quando aberto.

Editor em linha

Para facilitar a formatação do seu conteúdo de texto, o Editor de Postagem inclui um editor de texto embutido; destaque qualquer pedaço de texto no editor e uma versão pequena da barra de ferramentas principal contendo as opções mais comumente usadas, como negrito ou sublinhado de texto e inserção de links, aparecerá. 

Inserção de Link Avançado

O editor inline fornece uma ferramenta de inserção de link mais robusta que inclui opções adicionais para ajustar o comportamento do link. Acesse-a destacando o texto que você deseja alterar para um link no editor de post e, em seguida, clicando no ícone de corrente na barra de ferramentas do editor inline que aparece acima do texto. Um pop-up aparecerá com as seguintes opções:

  • Select link type - Define o tipo de conteúdo ao qual você deseja criar um link; selecione entre um  link de postagem ,  um link de tag ,  um link de autor ,  um link de página inicial ,  um link personalizado ou um  arquivo do gerenciador de arquivos.
  • Post, Tag or Author name - Esta opção só está disponível quando a página Post, Tag ou Autor é selecionada na seção anterior. Todos os itens disponíveis no seu site Publii serão mostrados ao clicar na lista suspensa, mas você pode usar a caixa de pesquisa no topo da lista para filtrar itens se tiver muito conteúdo no seu site.
  • Custom link - Define a URL do link. Aparece somente quando o tipo de link Custom link é selecionado.
  • Link target - Define se o link redireciona a janela atual do navegador do usuário (opção Same window) ou abre em uma nova guia (opção New window).
  • Link label - Define o texto que deve ser exibido como um link no conteúdo; se você destacou o texto previamente, ele será adicionado automaticamente a este campo.
  • Link Anchor  - Define o alvo para uma âncora que está na mesma página que o link, permitindo que os usuários pulem para uma seção específica rapidamente. A lista suspensa exibirá todas as âncoras que foram adicionadas à página para seleção.
  • Link title attribute - Define o atributo de título HTML para o elemento de link. O texto inserido aqui aparecerá como uma dica de ferramenta quando o usuário passar o mouse sobre o link no site publicado.

Atalhos de Markdown

O Editor TinyMCE no Publii não inclui atualmente um editor de markdown completo, mas tem suporte para os atalhos mais comuns, o que ajudará você a criar o conteúdo da sua postagem mais rapidamente.

## Dois hashes antes de uma frase definem o texto como um elemento HTML H2, ou Título 2.
### Três hashes antes de uma frase criam um elemento HTML H3, ou Título 3.
#### Quatro hashes criam um elemento h3, ou Título 4.
##### Cinco hashes criam um elemento H5, ou Título 5.
###### Seis hashes criam um elemento H6, ou Título 6.
1.  O número '1' e um ponto final seguido de um espaço criam um elemento HTML OL, ou Lista Ordenada.
-  ,  *  Um asterisco ou hífen (também conhecido como traço) seguido por um espaço cria um elemento HTML UL (Lista Não Ordenada).
*text* Inserir asteriscos antes e depois de um bloco de texto deixará esse texto  em itálico .
**text** Inserir dois inícios antes e depois de um bloco de texto deixará esse texto  em negrito .

Inserindo títulos com Markdown

Os elementos de título em HTML (H1-H6) são usados ​​para criar títulos para seu conteúdo. Cada 'nível' de título é usado para mostrar onde um título específico está na hierarquia de postagens; por exemplo, o título principal de uma página ou artigo usará o título H1, enquanto cada subseção do artigo usará um título H2. Se cada seção tiver suas próprias subseções, elas geralmente usarão o próximo título abaixo; uma subseção sob um título H2 usará um título H3, e assim por diante. Isso cria uma estrutura para seu conteúdo que é facilmente compreensível para pessoas e computadores! No Publii, o título H1 principal é criado pelo campo 'Título' no editor de postagens, e os outros títulos podem ser adicionados manualmente conforme você cria seu conteúdo. Com atalhos de markdown, isso pode ser feito de forma muito rápida e fácil.

Para inserir um título usando markdown, você adiciona hashes (#) correspondentes ao título que você quer inserir antes do seu título. Então, para inserir um elemento Heading 2 (H2), você escreveria:

## My heading 2 title

...então aperte enter. O Post Editor converterá isso automaticamente para um elemento Heading 2 em HTML. Se você quiser inserir um elemento Heading 3 (H3), basta adicionar três hashes antes do título, e assim por diante.

Inserindo listas com Markdown

Existem dois tipos de listas que são comuns em HTML e na escrita em geral; listas ordenadas, ou listas com itens numerados, e listas não ordenadas, que usam marcadores em vez de números para cada item. Inseri-las usando markdown é extremamente rápido e fácil. Para uma lista não ordenada, digite um asterisco (*) ou um hífen/traço (-) seguido por um espaço antes do texto do primeiro item em uma lista. Quando você pressiona enter, o texto muda automaticamente para um item de marcador. Por exemplo, digitando:

* The first item in my unordered list.

...e pressionar a tecla enter adicionará um marcador antes do texto e criará automaticamente um segundo marcador para sua nova linha de texto, para que você possa adicionar o próximo item. Basta adicionar seu próximo item e pressionar enter novamente para criar um terceiro item, etc... Depois de adicionar todos os itens desejados, pressionar enter em um item em branco encerrará a lista.

Inserir listas ordenadas funciona exatamente da mesma forma, mas em vez de usar um asterisco ou hífen, você pode simplesmente começar a escrever a lista usando " 1.  ". O item será automaticamente formatado em uma lista. Depois de adicionar itens suficientes, basta pressionar enter em um item em branco para encerrar a lista.

Adicionar texto em negrito ou itálico com Markdown

O Markdown pode alterar automaticamente seu texto para negrito ou itálico ao envolver o texto no símbolo apropriado, neste caso um asterisco (*) para itálico e dois asteriscos (**) para negrito. Para deixar uma frase em negrito, você deve adicionar dois asteriscos antes e depois do texto. Para itálico, adicione um asterisco antes e depois do texto:

**This text will be bold**. 
*This text will be italic*. 
This text will be unchanged.

O Editor de Postagem mudará o texto assim que você pressionar a barra de espaço após adicionar a última estrela, para que você saiba imediatamente que funcionou.
