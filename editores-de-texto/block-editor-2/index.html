Tutorial de utilização do Publii

Markdown Editor

O  Markdown Editor  é o mais simples dos três editores disponíveis no Publii; o que falta em recursos compensa em velocidade. Usando símbolos de taquigrafia, como hashes, você pode inserir vários  elementos HTML  sem precisar se preocupar em abrir e fechar tags. Esse processo permite a criação rápida de conteúdo; ótimo se você estiver escrevendo conteúdo que não exija muitas opções expansivas, como documentação ou instruções. 

Ao escrever em HTML, cada parte do conteúdo deve ser envolvida em  tags HTML ; por exemplo, um parágrafo normal será envolvido em uma  tag de parágrafo :

<p>This is a paragraph</p>

Com o Markdown, as tags de parágrafo são inseridas automaticamente no conteúdo, e tags mais complexas, como  Cabeçalhos e  Links, são adicionadas usando atalhos. Quais tags HTML têm atalhos depende do editor de markdown usado; no caso do Publii, o mecanismo de Markdown é o SimpleMDE  que, como o nome sugere, oferece um mecanismo simples e rápido para escrita. Nas seções a seguir, abordaremos os vários atalhos de markdown disponíveis no Publii.

Adicionando um título de postagem

Ao abrir o Editor Markdown pela primeira vez para criar uma postagem, você verá um espaço em branco com dois campos disponíveis, um marcado como  Add Post Title e o outro marcado como  Write Here... . 

[imagem=postagem em branco do editor de markdown]

O  Título do Post  é necessário antes que o post possa ser salvo, então clique no campo e insira seu título o mais rápido possível. Você pode, é claro, alterá-lo a qualquer momento, mas adicionar um título antes vai lhe poupar qualquer problema de salvar depois. 

A seção marcada como Write Here é para o conteúdo principal da sua postagem, e abordaremos como os atalhos do Markdown podem ser usados ​​para facilitar a escrita desse conteúdo.

Atalhos de Markdown

Para usar esses atalhos, adicione o símbolo especificado no início da linha; isso garantirá que ele seja convertido corretamente para HTML. Alternativamente, você também pode usar o atalho de teclado especificado para inserir o símbolo necessário:

# (Ctrl+H)  Um único hash antes de uma frase define o texto como um elemento HTML H1, ou Título 1.
##  (Ctrl+HH) Dois hashes antes de uma frase definem o texto como um elemento HTML H2, ou Título 2.
### (Ctrl+HHH)  Três hashes antes de uma frase criam um elemento HTML H3, ou Título 3.
####  (Ctrl+HHHH)  Quatro hashes criam um elemento h3, ou Título 4.
#####  (Ctrl+HHHHH) Cinco hashes criam um elemento H5, ou Título 5.
######  (Ctrl+HHHHHH) Seis hashes criam um elemento H6, ou Título 6.
1. (Ctrl+Alt+L)  O número '1' e um ponto final seguido de um espaço criam um elemento HTML OL, ou Lista Ordenada.
- , +, * (Ctrl+L)  Um asterisco, um hífen (também conhecido como traço) ou um sinal de mais seguido por um espaço cria um elemento HTML UL (Lista Não Ordenada).
*text*  (Ctrl+I)  Inserir asteriscos antes e depois de um bloco de texto deixará esse texto  em itálico .
**text**  (Ctrl+B)  Inserir dois inícios antes e depois de um bloco de texto deixará esse texto  em negrito .
`code`  Inserir um  acento grave antes e depois de um bloco de texto o alterará para  código embutido ; isso estiliza o texto para que apareça em um estilo não formatado com um fundo colorido.
``code`` Inserir dois  acentos graves antes e depois de um bloco de texto mudará esse texto para  Código ; este tem um estilo semelhante ao  Código embutido , mas mantém a formatação, como quebras de linha para blocos de código maiores.
---READMORE---  'READMORE' cercado por três hífens em ambos os lados criará um  ponto Readmore , que define onde qualquer texto de pré-visualização deve parar. Os usuários precisarão clicar no  Readmore para serem levados à postagem principal para ler o conteúdo completo.
> (Ctrl+Q)  Um sinal de "maior que" seguido por um espaço criará um  elemento Blockquote  ; o texto inserido depois desse elemento (mas apenas na mesma linha) será estilizado como texto de citação maior.
[title](link URL)(Ctrl+K)   Cria um link; o texto na  seção Título definirá o texto do link, enquanto a URL na  seção URL do link definirá o destino para o qual o link deve direcionar o navegador.
~~text~~  (Ctrl+U)  Inserir dois  tils  antes e depois de um pedaço de texto o transformará em texto tachado .

Inserindo títulos com Markdown

Os elementos de título em HTML (H1-H6) são usados ​​para criar títulos para seu conteúdo. Cada "nível" de título é usado para mostrar onde um título específico está na hierarquia de postagens; por exemplo, o título principal de uma página ou artigo usará o título H1, enquanto cada subseção do artigo usará um título H2. Se cada seção tiver suas próprias subseções, elas geralmente usarão o próximo título abaixo; uma subseção sob um título H2 usará um título H3, e assim por diante. Isso cria uma estrutura para seu conteúdo que é facilmente compreensível para pessoas e computadores! No Publii, o título H1 principal é criado pelo campo "Título" no editor de postagens, e os outros títulos podem ser adicionados manualmente conforme você cria seu conteúdo. Com atalhos de markdown, isso pode ser feito de forma muito rápida e fácil. Você pode, é claro, adicionar vários elementos H1 se desejar, mas isso geralmente não é recomendado para fins de SEO.

Para inserir um título usando markdown, você adiciona hashes (#) correspondentes ao título que você quer inserir antes do seu título. Então, para inserir um elemento Heading 2 (H2), você escreveria:

## My heading 2 title

...então aperte enter. Este conteúdo será automaticamente convertido para um elemento Heading 2 em HTML. Se você quiser inserir um elemento Heading 3 (H3), basta adicionar três hashes antes do título, e assim por diante.

Inserindo listas com Markdown

Existem dois tipos de listas que são comuns em HTML e na escrita em geral; listas ordenadas, ou listas com itens numerados, e listas não ordenadas, que usam marcadores em vez de números para cada item. Inseri-las usando markdown é extremamente rápido e fácil. Para uma lista não ordenada, digite um asterisco (*) ou um hífen/traço (-) seguido por um espaço antes do texto do primeiro item em uma lista. Quando você pressiona enter, o texto muda automaticamente para um item de marcador. Por exemplo, digitando:

* The first item in my unordered list.

...e pressionar a tecla enter adicionará um marcador antes do texto e criará automaticamente um segundo marcador para sua nova linha de texto, para que você possa adicionar o próximo item. Basta adicionar seu próximo item e pressionar enter novamente para criar um terceiro item, etc... Depois de adicionar todos os itens desejados, pressionar enter em um item em branco encerrará a lista.

Inserir listas ordenadas funciona exatamente da mesma forma, mas em vez de usar um asterisco ou hífen, você pode simplesmente começar a escrever a lista usando " 1.  ". O item será automaticamente formatado em uma lista. Depois de adicionar itens suficientes, basta pressionar enter em um item em branco para encerrar a lista.

Adicionar texto em negrito, itálico ou tachado com Markdown

O Markdown pode alterar automaticamente seu texto para negrito ou itálico ao envolver o texto no símbolo apropriado, neste caso um asterisco (*) para itálico, dois asteriscos (**) para negrito e dois tils (~~) para tachado. Para deixar uma frase em negrito, você deve adicionar dois asteriscos antes e depois do texto. Para itálico, adicione um asterisco antes e depois do texto, e para tachado, dois tils antes e depois do texto:

**This text will be bold**. 
*This text will be italic*.
~~This text will be strikethrough~~
This text will be unchanged.

O Markdown Editor mudará o texto assim que você adicionar os símbolos antes de uma palavra, então você saberá imediatamente que funcionou. A  tecla Til geralmente está localizada no canto superior esquerdo de um teclado QWERTY padrão; se você estiver com problemas para encontrá-la, você pode usar o atalho de teclado para texto tachado  (Ctrl+U)  ou pode usar o  Alt  Code ; segure a  tecla Alt  , então pressione os números  126 , então solte a  tecla Alt  para inserir um til.

Inserindo Links com Markdown

Os links são fáceis de inserir no seu conteúdo com Markdown. Primeiro, você pode definir o texto do link usando colchetes  [ ]  ao redor da palavra com a qual deseja criar um link e, em seguida, colchetes regulares  ( )  ao redor do link ao qual deseja vincular:

[link text](link URL)

Você também pode criar um link pressionando as teclas  CTRL+K  ou  Command+K  no seu teclado para abrir o pop-up Inserir/Editar Link  . 

[imagem]Inserir pop-up de link no editor de markdown[/imagem]

Neste pop-up, a primeira opção que você verá é:

  • Select Link Type  - define o tipo de conteúdo ao qual você deseja criar um link; selecione entre um  link de postagem ,  um link de tag ,  um link de autor ,  um link de página inicial ,  um link personalizado ou um  arquivo do gerenciador de arquivos .

Esta opção define o tipo de conteúdo ao qual você deseja vincular.  Post ,  Tag e  Author Links permitem que você vincule a uma postagem, tag ou página de autor específica no seu site Publii.  Homepage Links apontam apenas para a homepage do site.  Custom Links apontam para uma URL que não faz parte do seu site Publii, e  File From File Manager permite que você vincule a um arquivo nos arquivos do seu site que os usuários podem visualizar ou baixar. 

Após selecionar o tipo de link que você deseja inserir, o próximo campo mudará dependendo do tipo de link. Para  Post ,  Tag  ou  Author  Links , o segundo campo conterá uma lista suspensa onde você pode escolher o  Post ,  Tag ou  Author  Page  que você deseja vincular. Se você tiver muitos itens na lista, você pode começar a digitar no campo para filtrar dinamicamente a lista.

Para  Custom Links , o segundo campo será usado para inserir a  URL  que você deseja vincular. Certifique-se de inserir o endereço completo para evitar criar um link quebrado.

Para  File From File Manager , o segundo campo funcionará de forma muito parecida com o  campo Post ,  Tag ou  Author Page  ; uma lista suspensa que contém todos os arquivos armazenados atualmente no gerenciador de arquivos do Publii. Novamente, você pode filtrar dinamicamente a lista digitando o nome do arquivo que deseja encontrar no campo.

Com o endereço do link definido, o campo final,  Link Label , permite que você defina o texto do link como ele aparecerá no seu conteúdo. Depois de preencher todos os campos, clique no  botão OK  para criar o link.

Inserindo código

Quando falamos sobre inserir código em nosso conteúdo, não queremos dizer código que será executado pelo navegador. Em vez disso, refere-se a uma maneira de estilizar o texto para que ele seja distinto e claro do resto do conteúdo da página; você pode vê-lo usado ao longo desta página ao demonstrar o que inserir no editor de markdown para alterar o estilo do texto, na verdade, vamos usá-lo agora para mostrar a você como inserir código em seu conteúdo:

`This text will be displayed as code content`

No entanto, há de fato dois tipos diferentes de conteúdo de código que podem ser inseridos, aos quais nos referimos como  Inline Code  e, simplesmente,  Code . A diferença é mínima na prática, mas importante para usuários que desejam inserir blocos de código maiores. 

Inline Code , que é inserido como no exemplo acima, é usado para exibir uma única linha de código; o comprimento não importa, mas o conteúdo será quebrado em uma nova linha. Para inserir  Inline Code , precisamos adicionar um  acento grave  antes e depois do texto do nosso código:

`This is an example of inline code in markdown; note that this long sentence wraps to a new line without any additional line breaks or other separators`

Para referência, o acento grave  não é um apóstrofo, embora pareça similar. Ele geralmente estará na tecla superior esquerda de um teclado QWERTY padrão, embora, dependendo do idioma que seu sistema operacional usa, você pode precisar segurar a tecla Shift  para acessá-lo. 

Ao contrário do código inline, a  inserção de código  manterá a formatação, como quebras de linha. Isso é particularmente útil ao exibir código que tem várias linhas separadas que não funcionarão corretamente se inseridas em uma única linha. Para criar um  bloco de código  , basta adicionar três  acentos graves antes ou na linha acima do texto, e outros três em uma linha separada após o final do texto que você deseja converter em um bloco de código:

```

This is an example of Code in markdown.

Line breaks, including blank lines, are maintained in this block.

This makes it easier users to read more extensive code listings.

```

Se você tiver dificuldade para encontrar a  tecla de acento grave  no seu teclado, você também pode digitá-la (na maioria dos sistemas operacionais europeus e americanos) mantendo pressionada a  tecla Alt ou  Command e pressionando os números  96 e, em seguida, soltando a  tecla Alt  . 

Inserindo Citações em Bloco 

Um  Blockquote  é um pedaço de texto estilizado para parecer maior com aspas; útil para chamar a atenção para uma citação importante do seu conteúdo ou para incluir uma citação de outra fonte que seja relevante para o seu conteúdo:

[imagem=bloco de citação exemplo]

Para inserir um blockquote, insira um  símbolo de maior que antes da linha que deseja converter em blockquote, seguido de um espaço:

> This will be the blockquote text

Pressionar Enter continuará o blockquote; o conteúdo adicionado em uma nova linha será quebrado com as linhas existentes para formar uma sequência contínua de texto:

> These two lines will appear
> as a single line of text in the blockquote

Para finalizar o blockquote, pressione Enter duas vezes, assim como você faria ao finalizar uma lista ordenada ou não ordenada.

Inserindo Imagens

Adicionar imagens ao seu conteúdo funciona de forma semelhante à adição de links no Markdown. Primeiro, você define a  descrição da imagem entre colchetes precedidos por um ponto de exclamação ![  ] . Em seguida, você define a imagem usando a URL da imagem entre colchetes regulares  ( ) . Por exemplo, para inserir uma imagem chamada  image1 e dar a ela a descrição 'picture' no texto, escreveríamos:

![picture](mytest.com/image1.png)

Alternativamente, você pode arrastar e soltar  imagens do seu computador diretamente no Markdown Editor no Publii, e ele adicionará automaticamente a imagem aos arquivos do site e criará o link para exibir a imagem. Neste ponto, você pode alterar o texto 'descrição da imagem' para seu próprio texto de descrição.
