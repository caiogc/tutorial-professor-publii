Tutorial de utilização do Publii

Tags

Em vez de categorias padrão que você pode estar acostumado, o Publii usa Tags , que são palavras-chave que se aplicam a um post. Um post pode ter várias tags anexadas a ele, então mesmo que seu post cubra muitos assuntos, os leitores conseguirão encontrá-lo.

Para criar uma nova tag, clique na opção Tags no menu da barra lateral esquerda. Se você não tiver nenhuma tag, verá uma página simples com um grande botão Add New Tag , então clique nele e a barra lateral Add New Tag  aparecerá à direita da tela.

O  menu da barra lateral Adicionar nova tag  será aberto, onde você poderá inserir os detalhes da nova tag e concluir a criação.

Opções de tags

As opções disponíveis ao adicionar ou editar uma tag são separadas em grupos com diversas opções em cada um, conforme a seguir:

Informações básicas

Esta seção abrange os elementos básicos para a Tag. As opções disponíveis nesta seção são: 

  • Name - Insira o nome da sua tag aqui.
  • Description - Coloque uma descrição para sua tag aqui, como o tipo de conteúdo ao qual ela está associada. O texto inserido aqui será exibido na página de tag individual em seu site, que lista todas as postagens em uma tag específica.
  • Hide tag - Esta opção permite ocultar uma tag de todas as listagens de postagens e páginas de postagens individuais; isso significa que a página de tags específica para a tag oculta não será renderizada para reduzir o peso do seu site.

Imagem em destaque

A imagem em destaque de uma tag é a imagem que será adicionada no topo da página da Tag, ou sempre que uma tag for exibida em uma vitrine, caso seu tema tenha tal recurso. As opções disponíveis são:

  • Featured Image - A imagem em destaque geralmente aparece no topo da página de tags e também é usada como imagem de pré-visualização de tags. Para adicionar uma imagem em destaque à sua tag, você pode arrastar e soltar uma imagem na caixa Imagem em Destaque  ou clicar no  botão Escolher Arquivo  para abrir uma janela do explorador de arquivos para selecionar uma foto do seu computador.
  • Alternative Text - Esta opção só está disponível se você tiver definido uma imagem em destaque. Aqui, você pode adicionar texto que será exibido se a imagem não puder ser carregada, ou lido por um leitor de tela usado por usuários com deficiência visual para descrever a foto.
  • Caption  - Outra opção somente para imagem em destaque, o texto adicionado aqui será adicionado como legenda abaixo da foto.
  • Credits  - O nome ou outro texto adicionado aqui aparecerá como crédito, permitindo que as pessoas saibam quem tirou a foto ou detém os direitos autorais, caso não sejam seus.

SEO

SEO, ou  Search Engine  Optimization , refere-se a opções que podem ser usadas para controlar como a página da sua Tag ficará ao aparecer nos resultados do mecanismo de busca, ou melhorar sua classificação nas pesquisas. As opções disponíveis são : 

  • Slug - O nome que você digitar aqui deve ser amigável a links; ou seja, não deve conter espaços ou caracteres especiais, pois será usado para links em seu site, por exemplo, se você tiver uma categoria chamada "Últimas Notícias", o slug pode ser "latest_news" . O slug será gerado automaticamente se este campo for deixado em branco.
  • Page Title - define o metatítulo (que é usado para fins de SEO e será exibido nos resultados dos mecanismos de busca); a indexação deve ser habilitada nas configurações do Publii antes que esta opção possa ser usada.
  • Meta Description - Defina a meta description, que funciona de forma similar ao meta title. Novamente, a indexação deve estar habilitada para usar esta opção.
  • Meta Robots Index  - Esta opção define o comportamento dos robôs dos mecanismos de busca (ou seja, os programas que os mecanismos de busca usam para coletar informações em sites e classificá-los em grupos para uso em pesquisas). Index/Noindex define se a página da web é indexada pelos mecanismos de busca para que ela possa aparecer nas listagens de pesquisa; Follow/Nofollow  define se os links na página devem ser seguidos pelos robôs.
  • Canonical URL - Define qual URL é a URL base, ou mais importante, para a página. Se deixado em branco, a URL da Tag Page padrão será definida como a URL canônica. 

Outras opções

Esta seção lida principalmente com a configuração de quais elementos da página de tags devem ser exibidos; por meio das opções nesta seção, você pode definir se, por exemplo, a imagem em destaque é exibida na página de tags ou não. A maioria das opções aqui são autoexplicativas, mas há uma opção que deve ser observada:

  • Custom Template - define o layout do modelo a ser usado pela página de tags; temas diferentes podem incluir apenas um modelo padrão ou várias alternativas.

Encontrando uma Tag

Conforme você adiciona novas tags, elas serão listadas na tela de abertura  Tags  que você vê ao clicar na  opção Tags na barra lateral esquerda. Por padrão, as tags são listadas da mais nova para a mais antiga, mas você pode facilmente alternar a ordem clicando no cabeçalho da coluna, por exemplo, clicando no cabeçalho ID  , você pode classificar suas tags por ordem de criação (conforme cada nova tag é criada, ela recebe um ID, começando em '1' para a primeira tag). 

Se você tem um grande número de tags e precisa encontrar uma específica, você pode usar a  opção Pesquisar  clicando no ícone de lupa que está ao lado do  título Tags  no topo da página. As pesquisas são dinâmicas e filtrarão a lista conforme você digita, então será fácil encontrar o que você está procurando.  

Excluindo uma Tag

Para excluir uma tag, na lista de tags que você vê pela primeira vez ao abrir a página de tags, clique na caixa de seleção à esquerda da tag que deseja excluir; assim que a marca de seleção for adicionada, a opção de exclusão aparecerá como um botão Delete na parte superior da seção:

Escondendo uma Tag

Para ocultar uma tag para que ela não fique visível no seu site (útil se você deseja classificar postagens em categorias para seu próprio rastreamento), você pode seguir os mesmos passos usados ​​para excluir uma tag; na seção de tags que exibe a lista de suas tags, clique na caixa de seleção à esquerda da(s) tag(ns) que deseja ocultar; uma opção para Ocultar (Hide) as tags aparecerá à direita do botão Excluir na parte superior da seção; clique aqui e as tags selecionadas serão ocultadas.

Editando uma Tag

Editar uma tag é basicamente o mesmo que adicionar uma tag, exceto que as opções já estão preenchidas! Para abrir a barra lateral Editar Tag, basta clicar no nome da tag que você deseja modificar.
