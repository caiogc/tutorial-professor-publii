Menus

Os menus são essenciais em qualquer bom site; você vai querer que os usuários consigam encontrar facilmente o caminho. O Publii tem como objetivo tornar a criação e o funcionamento dos seus menus uma tarefa fácil. Para chegar à tela Menus, clique na opção Menus na barra lateral esquerda.

Criando um Menu

Para criar um novo menu, clique no  botão Add New Menu  , que estará no centro da tela ou no canto superior direito, dependendo se você já tem algum menu ou não. Uma caixa pop-up será aberta; aqui você pode nomear seu menu. Uma vez feito, clique no  botão Create new menu  .

Seu menu agora está criado, mas há um problema; não há itens de menu para as pessoas clicarem! Na  tela Menus  , clique no seu novo menu para expandir suas opções. Como nenhum item foi adicionado ainda, você verá apenas dois botões;  Add Menu Item  e  Edit Menu Name .

Clique no  botão Add Menu Item  ; isso abrirá uma barra lateral no lado direito da tela, onde você pode fornecer detalhes para seu item de menu. Inicialmente, há seis opções disponíveis:

  • Label  - Este é o nome que será usado para o item de menu.
  • Type  - Define que tipo de link o item de menu será, por exemplo, um link para uma postagem ou página de tags, link externo etc. Mais informações sobre os tipos de link disponíveis estão na próxima seção.
  • Link Title Attribute  - define um atributo para o título do item de menu; isso é útil se você quiser que o item de menu use um conjunto específico de regras CSS ou o separe dos outros itens de menu.
  • CSS Class  - Adicione uma classe CSS ao menu para facilitar a estilização; necessário somente para usuários avançados que desejam adicionar seu próprio código CSS a um tema.
  • Link Target  - define se o link redirecionará a janela atual do usuário para o destino do link ou abrirá uma nova guia ou janela do navegador para carregá-lo; isso é particularmente útil ao criar links para URLs externas.
  • Link "rel" Attribute  - Define o  atributo de relacionamento  para o item de menu; por exemplo, definir isso como  nofollow  impedirá que robôs de mecanismos de busca sigam o link, o que é útil para fins de SEO.

Quando tiver todas as suas opções do jeito que você quer, clique no  botão Add menu item na parte inferior da barra lateral. O item de menu será adicionado abaixo do menu selecionado. Repita conforme necessário até que você tenha adicionado todos os links que precisa.

Tipos de itens do menu

A opção Type que você pode ver ao criar um novo item de menu é a configuração mais importante, pois define o comportamento do item; ou seja, o que acontece quando um visitante do site clica nele no menu. Os vários tipos permitem que você aponte para artigos específicos ou páginas de tags, ou vincule a recursos externos, bem como forneça algumas outras ações úteis para sites maiores ou mais complexos. Os tipos de itens de menu disponíveis são os seguintes:

  1. Post Link

    O tipo de item de menu Post Link faz o que você esperaria; ele vincula a uma das postagens no seu site Publii. Isso pode ser muito útil se você quiser criar uma página Sobre nós ou outra página informativa separada do seu outro conteúdo; basta criar a postagem que você deseja usar e vinculá-la no menu principal. Quando o tipo Post Link é selecionado, uma nova opção aparecerá à direita do campo Tipo:
    Página de postagem - Aqui você pode selecionar a postagem específica à qual deseja que o item de menu faça o link. Clicar no campo abrirá uma lista suspensa de seus artigos, com uma pequena barra de pesquisa no topo da lista que será muito útil se você tiver muitas postagens! A pesquisa é dinâmica, então inserir as primeiras letras do nome da postagem filtrará os itens instantaneamente.

  2. Tag Link

    O item de menu Tag Link tem a mesma função que o tipo Post Link , exceto que ele vincula a uma página de tag específica em vez de um post. Isso pode ser muito útil se você estiver criando um site de notícias e quiser que os leitores tenham acesso instantâneo às suas diferentes categorias de notícias, por exemplo. Quando esse tipo de item for selecionado no campo Type , uma nova opção aparecerá:
    Tag page - Aqui você pode definir a página de tags específica à qual deseja que o item de menu faça o link. Clicar na opção abrirá uma lista de suas páginas de tags disponíveis, ou você pode usar a barra de pesquisa dinâmica no topo da lista para filtrar os resultados; muito útil se você tiver muitas postagens!

  3. Tags List Link

    O item de menu Link da lista de tags permite que você crie um link para uma página de tags que exibe todos os artigos de uma tag específica.

  4. Author Link

    O item de menu Author Link permite que você crie um link para uma página de autor específica; conforme seu site se torna mais estabelecido, alguns usuários podem gostar de poder rastrear o trabalho de seu autor favorito! Quando esse tipo de item é selecionado entre os tipos disponíveis, uma nova opção aparecerá:
    Author page  - Assim como nas páginas de post e tag, você pode selecionar qual página do autor vincular a partir desta lista suspensa. Se você tiver muitos autores, pode começar a digitar o nome do autor para filtrar dinamicamente a lista.

  5. Homepage Link

    Um link tão simples quanto possível; os itens do menu Homepage Link vinculam de volta à sua página inicial, para que possam ser usados ​​como um botão Início.

  6. Internal Link

    O item de menu Link interno permite que você insira manualmente um endereço de link interno para seu site; isso pode ser útil se você tiver páginas criadas por plugins que não fazem parte do pacote principal do Publii ou outras páginas que não sejam de postagem/tag/autor. Quando esse tipo de item é selecionado, um campo de link interno vazio aparecerá, onde você pode inserir o URL do seu link interno.

  7. External Link

    O item de menu External Link permite que você insira manualmente uma URL externa para o item de menu para vincular. Isso pode ser usado para, por exemplo, vincular ao Paypal para doações ou apontar para sites parceiros. Quando esse tipo de item é selecionado, um novo campo External link aparecerá, onde você pode inserir a URL para a qual deseja que o item de menu seja vinculado.

  8. Text Separator

    O item de menu Text Separator é um item não funcional que é usado principalmente quando você tem muitos itens de submenu. Por exemplo, você pode ter um item Text Separator chamado Categories , que não é clicável, mas se os usuários passarem o mouse sobre ele, um submenu contendo itens Tag Link será aberto.

Atribuindo Menus

Os menus no Publii são colocados de acordo com seções predefinidas no tema; cada tema que você baixa e usa pode ter locais de menu completamente diferentes incorporados. Para atribuir um menu a um local de menu disponível no tema escolhido, clique na opção Menus na barra lateral esquerda e, em seguida, clique no botão suspenso abaixo da coluna Assigned Menu no lado direito da tela. Isso acionará um pop-up com os locais de menu disponíveis e a opção de nível máximo.

As opções do Menu Atribuído permitem que você exiba uma única instância de menu em vários locais em todo o seu site, eliminando a necessidade de criar menus duplicados. Por exemplo, você pode exibir seu menu principal nas seções de cabeçalho e rodapé do seu site. No entanto, essa funcionalidade depende do suporte integrado do seu tema para várias posições de menu. Você também pode definir o número de níveis de menu para cada local de menu específico.

Geralmente, o menu principal, localizado no topo de um site, exibe submenus, mas o menu de rodapé costuma ser mais conciso. Com a opção Max-level , você pode determinar o número de níveis de menu atribuídos para uma posição específica.

Lembre-se de que o número máximo de níveis de menu não pode exceder o valor definido no tema. Para permitir um número ilimitado de níveis de menu para uma posição, basta definir o valor como "-1"."

Criando itens de submenu

Para sites maiores, você pode adicionar submenus aos seus menus; por exemplo, se você quiser ter um  item de menu Sobre nós  que, quando clicado, abra um submenu com links para suas páginas informativas. 

Há duas maneiras de adicionar submenus a um item de menu: arrastando e soltando (mais informações sobre esse método podem ser encontradas na  seção Reestruturação de menus abaixo) ou você pode usar o  botão Add Submenu  que aparecerá ao lado de cada item de menu na lista.

Para adicionar um submenu usando este método, clique no menu que você deseja modificar, depois no  botão Add Submenu  ao lado do item ao qual você deseja adicionar o submenu. Isso abrirá a  barra lateral Add New Menu Item; preencha os campos para o item de menu normalmente, depois clique no  botão Add Menu Item na parte inferior da barra lateral. O novo item será adicionado à lista, mas será ligeiramente recuado para indicar que é um submenu do item de menu acima dele.  

Reestruturação de Menus

De tempos em tempos, você pode querer modificar a hierarquia de um dos seus menus. A tela Menus no Publii permite uma funcionalidade simples de arrastar e soltar para fazer isso. Se quiser, você pode criar sub-submenus! Basta arrastar e soltar seu item de sub-submenu abaixo e ligeiramente à direita de um item de submenu, e ele recuará novamente para mostrar que será colocado como um submenu de um submenu!

Reestruturação de menus por meio da opção Mover

Uma opção alternativa está disponível para reestruturar seus menus; o  botão Move. Esta opção é particularmente útil quando você precisa ser preciso ao mudar o layout do seu menu. 

Para usar a opção Mover, na lista de menus, clique no menu que você deseja modificar para expandir suas opções, então clique no comando Move  ao lado do item de menu que você deseja mover. O item de menu que você está movendo terá uma borda de linha tracejada adicionada a ele, e um novo conjunto de opções aparecerá nos outros itens de menu;  Before, After, e As Submenu.

Ao clicar na opção relevante, você pode mover o item selecionado para antes ou depois de um item de menu, ou adicioná-lo como um submenu. Na prática, isso pode parecer assim:

  1. Tenho dois itens de menu;  Item de Menu 1  e  Item de Menu 2 , e desejo tornar o Item de Menu 2  um subitem de menu para  o Item de Menu 1 .
  2. Ao clicar no  comando Mover  para  o Item de Menu 2 , ele será selecionado (indicado pela borda tracejada).
  3. Então, nas opções para  Item de Menu 1 , uma nova opção aparece; Inserir Item Selecionado . Aqui, posso dizer ao Publii exatamente onde colocar o item de menu clicando no texto relevante; Antes , Depois ou Como Submenu .
  4. Como desejo adicionar  o Item de Menu 2 como um Submenu do Item de Menu 1 , posso clicar na opção Como Submenu ao lado do Item de Menu 1 ; o item de menu será movido para esse novo local.
  5. Se eu decidir não mover o item, posso clicar no  texto vermelho Desmarcar ao lado do Item de Menu 2 para cancelar a movimentação. 

Ocultando um item de menu

Para habilitar ou desabilitar itens de menu no front-end, use o botão Hide. Note que ocultar um item de menu oculta todos os itens de menu filho.

Duplicando um item de menu

Cada item de menu, incluindo quaisquer itens de submenu, pode ser duplicado clicando no  ícone Duplicate  à direita do item de menu, ao lado do  ícone Garbage  ; parece dois quadrados sobrepostos. Ele será duplicado instantaneamente com um clique, então não clique com muito entusiasmo ou você pode acabar com muito mais itens duplicados do que precisa!

Excluindo um Menu ou Item de Menu

Para excluir um item de menu, clique no ícone Lixeira no lado direito do item de menu. Para excluir um menu inteiro, adicione uma marca de seleção à caixa de seleção à esquerda do menu que você deseja excluir; uma nova opção Delete Menu aparecerá no topo da seção, então clique no botão para confirmar a exclusão.
